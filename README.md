# 📚 Qt Kütüphane Otomasyonu

C++ ve Qt Framework kullanılarak geliştirilmiş masaüstü kütüphane yönetim sistemi.

Bu uygulama, kütüphanelerde gerçekleştirilen temel işlemleri dijital ortamda yönetmek amacıyla geliştirilmiştir. Sistem üzerinden kitap kayıtları, üye işlemleri ve ödünç alma/teslim etme süreçleri kolayca takip edilebilmektedir.

---

## 🚀 Proje Özellikleri

### 📖 Kitap İşlemleri

* Yeni kitap ekleme
* Kitap bilgilerini görüntüleme
* Kitap kayıtlarını düzenleme
* Kitap silme işlemleri

### 👤 Üye İşlemleri

* Yeni üye kaydı oluşturma
* Üye bilgilerini listeleme
* Üye kayıtlarını güncelleme
* Üye silme işlemleri

### 🔄 Ödünç İşlemleri

* Kitap ödünç verme
* Teslim edilen kitapları sisteme işleme
* Ödünç kayıtlarını takip etme

### 🖥️ Arayüz

* Qt Designer ile tasarlanmış kullanıcı dostu arayüz
* Çok pencereli yapı
* Görsel destekli menüler
* Masaüstü uygulaması deneyimi

---

## 🛠️ Kullanılan Teknolojiler

* C++
* Qt Framework
* Qt Creator
* Qt Designer
* Qt Resource System (QRC)

## ⚙️ Kurulum

### Gereksinimler

* Qt Creator
* Qt Framework
* C++ Derleyicisi (MinGW / MSVC)

### Çalıştırma

1. Projeyi klonlayın:

```bash
git clone https://github.com/KULLANICI_ADIN/qt-kutuphane-otomasyonu.git
```

2. Qt Creator ile açın:

```text
ODEV2KUTUPHANE.pro
```

3. Projeyi derleyin ve çalıştırın.

## 🎯 Projenin Amacı

Bu proje, nesne yönelimli programlama (OOP), Qt Framework kullanımı ve masaüstü uygulama geliştirme konularında deneyim kazanmak amacıyla geliştirilmiştir.

Proje kapsamında;

* Çok pencereli uygulama geliştirme,
* Qt Designer kullanımı,
* Kaynak yönetimi,
* Kullanıcı arayüzü tasarımı,
* Modüler yazılım geliştirme

konularında çalışmalar gerçekleştirilmiştir.

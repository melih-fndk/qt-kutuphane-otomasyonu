# Qt Kütüphane Otomasyonu

Bu proje, C++ ve Qt Framework kullanılarak geliştirilmiş masaüstü kütüphane otomasyon sistemidir.

## Kullanılan Teknolojiler

- C++
- Qt Framework
- Qt Designer

## Özellikler

- Kitap işlemleri
- Üye işlemleri
- Ödünç alma işlemleri
- Ödünç teslim etme işlemleri
- Çok pencereli arayüz

## Kurulum

Projeyi Qt Creator ile açmak için:

```bash
ODEV2KUTUPHANE.pro
